# for循环

⚜⚜⚜循环语法 :

1.wx:for="" 格式注意：wx:for=""需要加大括号

2.下标index和成员item内置的变量可以用

<!-- for循环 语法：wx:for -->

<!-- 循环语法 wx:for -->

<!-- 格式注意：wx:for="{{}}"需要加大括号 -->
<view wx:for="{{list}}">{{index}}=====>{{item}}</view>

替换默认变量：

<view wx:for="{{list}}"
  wx:for-index="a"
  wx:for-item="elementOS"
>{{a}}=====>{{elementOS}}</view>

⚜⚜⚜循环的时候：唯一的标识

语法：wx:key='?' 格式没有{{}} 可以放入数组成员内唯一标识id；如果实在找不到唯一标识，可以使用保留关键字*this ⚜⚜⚜注：*this不用加大括号

  // 有唯一值
<view class="lin"></view>
<view wx:for="{{arr}}" wx:key="id">{{item.name}}</view>

  // 没唯一值
<view wx:for="{{arr}}" wx:key="*this">{{item.name}}</view>
